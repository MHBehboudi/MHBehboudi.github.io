About

I'm a Ph.D. candidate in Cognition & Neuroscience at UT Dallas with an Electrical Engineering background. My dissertation maps how children's large-scale brain networks develop during language processing using EEG connectivity and graph theory.

I've developed a novel cross-frequency connectivity algorithm (MDPC), won an international neural decoding competition using foundation model architectures, and built automated pipelines processing 500+ GB of neural data. My work sits at the intersection of signal processing, machine learning, and experimental neuroscience.

Research Highlights

🏆 1st Place — LibriBrain Long Acquisition Track

Foundation Models for Neural Decoding of Speech

Designed an end-to-end deep learning pipeline leveraging self-supervised pretraining on unlabeled MEG data, followed by supervised fine-tuning with 1D-CNNs, Transformer encoders, and Graph Neural Networks to classify speech vs. silence from high-dimensional recordings. Achieved Macro-F1 = 0.85 against international teams.

PyTorch · Self-Supervised Learning · Transformers · GNNs · SLURM HPC
View on GitHub →

Novel Cross-Frequency Connectivity Algorithm

Developed Time-Lagged Cross-Frequency Multidimensional Pattern Connectivity (MDPC) to characterize dynamic information flow across brain networks during speech perception in noise. Full tri-language pipeline in Python, MATLAB, and R.

MDPC · Morlet Wavelets · PLV/wPLI · Graph Theory · Mixed-Effects Models
View on GitHub →

AI-Driven Sleep Apnea Diagnosis

Built a computer-aided diagnostic system using complexity features (fractals, Hurst exponent, entropy measures) with mRMR feature selection and SVM/KNN classification from single-channel clinical EEG. Two peer-reviewed publications.

EEG · Entropy · mRMR · SVM · 95% Accuracy
